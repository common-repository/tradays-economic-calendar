Tradays economic calendar plugin for WordPress websites

== Description ==

The official plugin for integrating Tradays economic calendar with WordPress websites.

= What is Tradays? =

Tradays is a free calendar of macroeconomic events for fundamental market analysis. Therein you will find financial news and indicators for the largest global economies - from the US and the EU to Australia and Japan. More than 800 macroeconomic indicators and events are collected from public sources in real time. Historical, current and forecast values, as well as importance characteristics are available for each indicator. Furthermore, the service stores extensive historical data outlined in charts and tables.

Add the plugin in your website and provide your audience with a free tool for monitoring economic events.

= Tradays advantages =

- No licensing risks: all data are collected automatically from primary sources by a custom software.
- Automatic update after a new event appears in a source.
- No ads.
- Support for 9 languages: all events and descriptions are translated into the most common global languages.
- It is free - we allow usage of the calendar and installation of the plugin in a website free of charge.
- The service is constantly expanding - new countries and languages are added regularly.
- It can be quickly installed in a website.

For more information, visit [the official Tradays website](https://www.tradays.com/?utm_source=wordpress.market&utm_medium=link&utm_term=tradays.website&utm_content=tradays.plugin.wordpress&utm_campaign=tradays.wordpress).

== Installation ==

To install the plugin, please follow these steps:

1. Upload plugin files to the directory `/wp-content/plugins/tradays-economic-calendar`
1. Activate the plugin in the 'Plugins' section of the WordPress console
1. Add a widget to the page in the 'Appearance' > 'Widgets'

== Frequently Asked Questions ==

= What is Tradays? =

Tradays is an economic calendar for fundamental analysis of financial markets. It provides over 900 indicators and events of the 15 largest global economies - from the US and the EU to Brazil and Japan. Find out more information about the service on [the official Tradays website](https://www.tradays.com/?utm_source=wordpress.market&utm_medium=link&utm_term=tradays.website&utm_content=tradays.plugin.wordpress&utm_campaign=tradays.wordpress).

= Is it free? =

Yes, the calendar is a completely free solution. You can embed it into any website.

= What are event sources? =

All calendar data are collected in real time from public sources - central banks, treasuries, ministries of finance, national bureaus and institutions of statistics, among other government agencies.
The entire calendar data is provided by MetaQuotes Ltd., owner of the [www.tradays.com](https://www.tradays.com/) website. [Terms of use](https://www.tradays.com/en/terms?utm_source=wordpress.market&utm_medium=link&utm_term=tradays.website&utm_content=tradays.plugin.wordpress&utm_campaign=tradays.wordpress).

= How often is the data updated? =

The calendar is updated immediately after a new event appears in a source.
